================================================================================
                      BACKEND INTEGRATION STEPS
================================================================================

When ready to add backend:

1. Set up Node.js/Express server
2. Configure MongoDB database
3. Create API endpoints (/api/*)
4. Replace lib/data.js calls with API fetch() calls
5. Implement JWT authentication
6. Add password hashing (bcrypt)
7. Set up payment gateway (Stripe/Razorpay)
8. Configure file uploads (AWS S3)
9. Deploy backend server
10. Update API URLs in frontend

See backend/README.md and ARCHITECTURE.md for detailed instructions.
